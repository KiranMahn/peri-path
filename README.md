## Get started

1. Clone this project 

2. cd into this project 

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the app

   ```bash
    npx expo start
   ```
5. Make sure the expo app is installed on your phone 

6. scan the QR code with your phone and the app will open in the expo app
